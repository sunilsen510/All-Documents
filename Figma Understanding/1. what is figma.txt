1. Figma क्या है?

Figma एक cloud-based design tool है जो UI/UX designers, web designers, mobile app designers और product designers के लिए use होता है।

यह tool आपको website, mobile apps, prototypes, wireframes और UI components design करने की सुविधा देता है।

मुख्य विशेषताएँ:

Cloud-based: कोई installation नहीं, browser में चल जाता है।

Real-time collaboration: जैसे Google Docs, multiple users एक ही project पर साथ काम कर सकते हैं।

Cross-platform: Windows, Mac, Linux, Chrome OS, और mobile apps पर भी use कर सकते हैं।

Prototyping: बिना coding के interactive prototypes बना सकते हैं।

Vector-based design: scalable designs बना सकते हैं।

7. Interview Questions & Answers (Figma Related)

Q1. Figma क्या है और इसे क्यों use करते हैं?
A: Figma एक cloud-based design tool है। इसे UI/UX design, prototypes, wireframes और collaboration के लिए use करते हैं।

Q2. Figma और Adobe XD में क्या difference है?
A:

Figma cloud-based है, real-time collaboration support करता है।

Adobe XD local install होता है और collaboration के लिए extra steps चाहिए।

Q3. Figma में Component और Instance में क्या difference है?
A:

Component: Master reusable element।

Instance: Component का copy जो master से linked रहता है।

Q4. Prototype बनाते समय क्या steps follow करते हैं?
A: Frames select → Prototype tab → Link screens → Interaction set → Preview → Share।

Q5. Figma में Auto Layout क्या है?
A: Auto Layout dynamic spacing, padding और alignment automatically manage करता है।

Q6. Figma के advantages और disadvantages क्या हैं?
A:

Advantages: Cloud, collaboration, cross-platform, reusable components।

Disadvantages: Offline use limited, advanced animations limited।

ठीक है! मैं आपको step-by-step बताता हूँ कि आप ऊपर बताए गए concepts और diagrams को Figma में practically कैसे use करेंगे।

Step 1: Figma Account और File Setup

https://www.figma.com
 पर जाएँ।

Free account बनाएँ या login करें।

Dashboard → “New File” पर click करें।

Canvas खुल जाएगा, जहाँ आप design करेंगे।

Step 2: Frames बनाना (Page/Layout)

Frame Tool shortcut: F

Canvas पर click + drag करके screen size select करें।

Desktop: 1440x1024

Mobile: 375x812

Frame को rename करें जैसे: Login Screen, Home Screen

Step 3: Shapes और Text Add करना

Rectangle Tool (R) → Buttons, cards, input fields बनाएँ

Text Tool (T) → Headings, labels, placeholder text डालें

Color / Typography:

Right panel → Fill / Stroke / Font settings

Example: Login Button → Rectangle + Text “Login”

Step 4: Components और Instances Create करना

Button या Navbar बनाएं → select करें → Right Click → “Create Component”

Components panel में saved रहेगा

Canvas पर copy paste करके Instances बनाएं

Master Component में बदलाव → सभी instances auto update

Step 5: Auto Layout (Dynamic spacing)

Buttons या menu items select करें

Right panel → “Auto Layout”

Direction: Horizontal / Vertical, Spacing: 16px, Padding set करें

Resize frame → elements dynamically adjust होंगे

Step 6: Prototype बनाना (Interactive Screens)

Login Button select करें

Right panel → Prototype tab → Drag connection → Home Screen

Interaction: On Click → Navigate To → Home Screen

Play button → Interactive preview

Step 7: Share & Collaborate

Top-right → “Share” button → Link copy करें

Team members या clients को link भेजें

वे comments add कर सकते हैं

Tips for Practical Use

Real projects में start from Wireframe → UI Design → Prototype → Feedback

Use plugins: Icons8, Unsplash, Charts

Components maintain करें → Consistency और fast updates
