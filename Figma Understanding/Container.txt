ЁЯз▒ 1. Container kya hai?

container ek fixed-width layout box hota hai.

Iska matlab тАФ iska width screen ke size ke according limit hota hai (responsive breakpoints par).

Jaise hi screen chhoti ya badi hoti hai, Bootstrap automatically uska width adjust karta hai тАФ

but har screen size ke liye ek maximum width hoti hai.

Example:

<div class="container">
  <h1>Fixed Width Layout</h1>
</div>


Result:

Large screen par center me ek fixed width box milega

Left-right me thoda margin (space) aayega

Content edge-to-edge nahi jayega

Bootstrap default max-width values (approx):

| Breakpoint    | Max width |
| ------------- | --------- |
| sm (тЙе576px)   | 540px     |
| md (тЙе768px)   | 720px     |
| lg (тЙе992px)   | 960px     |
| xl (тЙе1200px)  | 1140px    |
| xxl (тЙе1400px) | 1320px    |


ЁЯМК 2. Container-fluid kya hai?

container-fluid ek full-width container hota hai.
Yani yeh screen ke poore width (100%) tak stretch karta hai тАФ
chaahe screen chhoti ho ya badi.

Example:

<div class="container-fluid">
  <h1>Full Width Layout</h1>
</div>


Result:

Yeh poori screen (viewport) ke width tak phail jaayega

Koi side margin nahi hota

Mostly use hota hai headers, footers, background color sections ya banners ke liye

ЁЯФН 3. Visual Difference

| Property           | `.container`                                                              | `.container-fluid`  |
| -------- | ----------------------------- | ---------------------------------------------------------------|
| Width                                   | Fixed (changes by breakpoint)                     | Always 100%                     |
| Margins                              | Auto (centered)                                                       | None                                       |
| Use Case                             | Normal page content                                             | Full-width sections (headers, footers, etc.) |

ЁЯТб 4. Ek Practical Example

<!-- Fixed width section -->
<div class="container bg-light">
  <h2>Container - Centered and limited width</h2>
</div>

<!-- Full width section -->
<div class="container-fluid bg-dark text-white">
  <h2>Container Fluid - Full width background</h2>
</div>


ЁЯза Summary:

container = Responsive fixed width (center aligned content)

container-fluid = Full 100% width (edge-to-edge content)

ЁЯз▒ 2. .container kya karta hai?

ЁЯСЙ .container ek fixed-width box hota hai.

ЁЯТ╗ Desktop par: center me fixed width box ban jayega

ЁЯМК 3. .container-fluid kya karta hai?

ЁЯСЙ .container-fluid ek full-width box hota hai.
