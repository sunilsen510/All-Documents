HTML

CSS (style.css)
body {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  min-height: 100vh;
  font-family: Arial, sans-serif;
  background: #f0f0f0;
  gap: 50px;
}

/* Button hover animation */
.animated-button {
  padding: 15px 30px;
  font-size: 18px;
  background: #3498db;
  color: white;
  border: none;
  border-radius: 8px;
  cursor: pointer;
  transition: transform 0.3s ease, background 0.3s ease;
}

.animated-button:hover {
  background: #2ecc71; /* Color change */
  transform: scale(1.1); /* Slight zoom effect */
}

/* Image scaling effect */
.animated-image {
  width: 200px;
  height: 200px;
  border-radius: 12px;
  transition: transform 0.4s ease, box-shadow 0.4s ease;
}

.animated-image:hover {
  transform: scale(1.2); /* Zoom in */
  box-shadow: 0 10px 20px rgba(0,0,0,0.3); /* Shadow effect */
}

Logic / Explanation

Button Animation:

transition: transform 0.3s, background 0.3s; ‚Üí Smooth animation

Hover ‚Üí scale(1.1) ‚Üí slightly bigger

Hover ‚Üí background color change ‚Üí attention grab

Image Animation:

transform: scale(1.2) ‚Üí zoom in

box-shadow ‚Üí depth effect

Smooth because transition applied

Performance:

Only transform + background + box-shadow ‚Üí GPU accelerated

Avoid layout-changing properties like width/height

‚úÖ Advanced / Logic-Based Tip

Combine multiple animations:

.animated-button:hover {
  transform: scale(1.1) rotate(-3deg);
  background: #2ecc71;
}


Stagger effects for multiple elements:

.container button:nth-child(1){ transition-delay:0s; }
.container img:nth-child(2){ transition-delay:0.1s; }
