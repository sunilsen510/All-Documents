1️⃣ Inline Elements
Definition (परिभाषा)

Inline elements केवल अपने कंटेंट (text या image) की चौड़ाई और हाइट लेते हैं।

ये अपने आप नई लाइन में नहीं जाते।

Common examples: <span>, <a>, <strong>, <em>, <b>, <i>

Properties

Width और Height सेट नहीं कर सकते (default behavior में)।

Padding और Margin केवल horizontal (left/right) में असर करती है, vertical (top/bottom) में कम असर दिखता है।

Inline elements text की तरह flow करते हैं।

Example
<p>यह एक <span style="color:red;">Inline</span> element है।</p>


Explanation:
<span> केवल text के हिस्से पर असर करेगा, नया line नहीं बनाएगा।

2️⃣ Block Elements
Definition (परिभाषा)

Block elements पूरी लाइन ले लेते हैं और नई लाइन पर start होते हैं।

Width और Height explicitly set कर सकते हैं।

Common examples: <div>, <p>, <h1>-<h6>, <section>, <header>, <footer>

Properties

Width default: 100% parent container की।

Height default: content के हिसाब से auto।

Padding, margin सभी sides में apply होते हैं।

Example
<div style="background-color:lightblue; width:200px; height:100px;">Block Element</div>
<p>यह नया line पर है।</p>


Explanation:
<div> पूरी line लेगा, उसके बाद नया line शुरू होगा।

3️⃣ Inline-Block Elements
Definition (परिभाषा)

Inline-Block elements inline की तरह line में आते हैं, लेकिन block की तरह width और height set कर सकते हैं।

Essentially ये inline और block का mixture हैं।

Common examples: <img>, <button>, <input>, या किसी element को display:inline-block देने पर।

Properties

Inline की तरह line में आते हैं।

Block की तरह width, height, padding, margin full apply होते हैं।

Useful for layout without breaking lines।

Example
<span style="display:inline-block; width:150px; height:50px; background-color:orange;">
Inline-Block Element
</span>
<span style="display:inline-block; width:150px; height:50px; background-color:green;">
Another Inline-Block
</span>


Explanation:
दो span एक ही line में रह सकते हैं और अपनी width/height ले सकते हैं।

5️⃣ When, Why, Where, How to use
Inline

When: Text के parts को style करने के लिए।

Why: Full line नहीं चाहिए, केवल text पर apply करना है।

Where: Links, highlights, emphasis।

How: <span>, <a> या display:inline;

Block

When: Containers, sections, headings, paragraphs।

Why: Full line layout चाहिए।

Where: Page sections, wrappers, forms।

How: <div>, <section> या display:block;

Inline-Block

When: Multiple small blocks line में चाहिए।

Why: Width/height adjustable और line में रहना है।

Where: Buttons, menus, cards in same row.

How: display:inline-block;

6️⃣ Interview Questions & Answers

Q1. Inline और Block elements में क्या difference है?
A: Inline elements line में रहते हैं, width/height set नहीं कर सकते। Block elements new line लेते हैं, width/height set कर सकते हैं।

Q2. Inline-Block क्यों use करते हैं?
A: अगर हमें elements line में रखने हैं, लेकिन उनका width/height customize करना है।

Q3. क्या <span> को block बनाया जा सकता है?
A: हाँ, CSS में display:block; देकर।

Q4. Block element को inline क्यों नहीं कर सकते?
A: हाँ, display:inline; देकर किया जा सकता है, लेकिन width/height control नहीं रहेगा।

Q5. Padding और Margin का Inline और Inline-Block में difference?
A: Inline में vertical margin/padding काम नहीं करता। Inline-Block में सभी sides काम करते हैं।

Q6. एक practical scenario बताइए जहाँ Inline-Block use करना सही है।
A: Navigation menu में, जहाँ menu items एक line में होने चाहिए और height/width uniform हो।
