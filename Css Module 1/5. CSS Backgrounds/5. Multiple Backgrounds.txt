üîπ Step 4: Real-World Interview Questions You Can Expect

Q1. Why did you use multiple backgrounds instead of multiple HTML layers?
A:

Keeps DOM clean ‚Üí better performance

Easier maintenance

Layer control (gradient + image + pattern)

Q2. How does background-attachment: fixed affect mobile devices?
A:

Not fully supported on some mobile browsers

Use JS-based parallax library for full support or fallback to scroll

Q3. Why use gradient overlay?
A:

Improves text readability over bright images

Maintains design consistency

Q4. How do you ensure images are responsive?
A:

Use background-size: cover and background-position: center

Use media queries to switch to optimized mobile images

Q5. How would you explain stacking order of multiple backgrounds?
A:

First background ‚Üí topmost layer

Last background ‚Üí bottom layer

CSS renders layers from top to bottom
