‚öñÔ∏è Pseudo-Classes vs Pseudo-Elements
Feature	Pseudo-Class	Pseudo-Element
Target	Element state or position	Specific part of element / content
Example	:hover, :focus, :first-child	::before, ::after, ::first-letter
Function	Conditional styling	Decorative / special content styling
DOM	No extra content	Virtual content rendered by browser


| Feature  | Pseudo-Class                   | Pseudo-Element                       |
| -------- | ------------------------------ | ------------------------------------ |
| Target   | Element state or position      | Specific part of element / content   |
| Example  | `:hover, :focus, :first-child` | `::before, ::after, ::first-letter`  |
| Function | Conditional styling            | Decorative / special content styling |
| DOM      | No extra content               | Virtual content rendered by browser  |
